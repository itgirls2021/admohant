The Importance of Cryptography

Adya Mohanty / July 20th, 2021

Cryptography is an important part of computer science that allows data transactions to occur securely on the internet. Overall, cryptography is a positive study because it is used to secure so many processes such as taking money out of an ATM, sending a text message, and file storage. It was really interesting to learn about the history behind cryptography and how complicated messages were encrypted and decrypted by hand, and how now that tedious work has been streamlined thanks to modern technology.
